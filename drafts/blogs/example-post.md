# Welcome to Hatamu Labs

This is an example blog post to demonstrate the markdown blog system. You can write your blog posts in markdown format and they'll be automatically converted to HTML.

## Features

Here are some features of this blog system:

- **Markdown support**: Write in simple markdown format
- **Automatic publishing**: Just add your `.md` file to `drafts/blogs/` and run the generator
- **Clean design**: Minimal, readable layout
- **Code highlighting**: Support for inline `code` and code blocks

## How to Write

Simply create a new markdown file in the `drafts/blogs/` directory with the following format:

```markdown
---
title: Your Blog Title
date: 2026-01-23
---

# Your Blog Title

Your content goes here...
```

## Publishing

Run the following command to publish your drafts:

```bash
node generate-blogs.js
```

That's it! Your blog posts will be published to the website.
